## Supported Ruby versions
* Ruby 1.8.7 or later
* jruby
* rubinius 2.1.1 or 2.2.3 (or later, maybe)
  * 2.2.1 and 2.2.2 are not working.
* (Possibly) ironruby / macruby / topaz etc
